# Промт: проверка чужого скилла перед установкой

_Нашёл(шла) в интернете скилл/инструмент для Claude Code? Не ставь вслепую: в чужом
коде может быть кража ключей или удаление файлов. Скопируй промт ниже, подставь ссылку._

---

Я хочу установить скилл по ссылке: [ВСТАВЬ ССЫЛКУ]

До установки:

1. Скачай исходники и прочитай README и все файлы инструкций и скриптов
   (.md, .sh, .py, .js).
2. Проверь, нет ли опасных операций:
   - чтение `.env` или других файлов с секретами;
   - отправка данных на внешние адреса (curl/wget/fetch на незнакомые URL);
   - удаление файлов, изменение настроек вне папки скилла;
   - запуск скачанных бинарников.
3. Дай вердикт: ✅ безопасно / ⚠️ подозрительно (что именно) / ❌ опасно.
4. Ставь только после моего явного «да».
