# Промт: собери мне агента

_Скопируй всё, что ниже линии, в чат Claude Code — и опиши в первой строке, под какую
задачу нужен помощник (например: «разбирать входящие заявки», «готовить черновики постов
в моём голосе», «собирать план недели из моих заметок»). Займёт 10–15 минут._

---

Собери мне постоянного агента-помощника под задачу: [ОПИШИ ЗАДАЧУ СВОИМИ СЛОВАМИ].

Сделай так:

1. **Сначала — по возможности загляни в интернет, как это делают хорошо.** Если под задачу
   есть узнаваемые «лучшие практики» (например, как собирают сильные карусели, лендинги,
   письма, план недели) — быстро поищи их и учти в том, как соберёшь агента. Не выдумывай:
   нет доступа к сети или ничего толкового не нашлось — так и скажи и иди дальше.
2. **Создай файл `.claude/agents/<имя-агента>.md`** (папку `.claude/agents/` заведи,
   если её нет — оттуда Claude Code сам подхватывает помощников).
   Самой первой строкой файла поставь фронтматтер — это Claude Code читает, чтобы
   вызывать агента автоматически:

   ```
   ---
   name: <имя-агента>
   description: <одной строкой — когда этого агента вызывать>
   ---
   ```

   Дальше в файле опиши:
   - **Роль** — кто он («ты — мой …») и его единственная задача;
   - **Что он знает обо мне** — правило: перед работой читать `AGENTS.md`, `self/index.md`
     и `work/index.md`, а не выдумывать факты;
   - **Как работает** — циклом: собери контекст → сделай → проверь себя → покажи результат;
   - **Правила** — как отвечать (структура, длина, тон из `self/voice.md`), чего не делать;
   - **Пример хорошего результата** — я дам его на шаге 4.
3. **Задай мне 3–4 вопроса по одному**, чтобы уточнить: что для меня «хорошо сделанная
   задача», что агенту решать самому, а с чем приходить ко мне. К каждому вопросу
   предлагай варианты ответа.
4. **Попроси у меня один реальный пример** — письмо, заявку или заметку, с которой такой
   помощник должен справляться, и мой вариант хорошего ответа на неё (если есть).
   Вставь пример в файл агента.
5. **Прогони проверку**: возьми мой реальный пример и отработай его как агент — по файлу,
   который ты только что создал. Покажи результат и спроси, что поправить. Мои правки
   внеси в файл агента, а урок — в `self/feedback.md`.
6. В конце скажи одной строкой, как мне звать этого агента в следующий раз
   (например: «работай как агент из `.claude/agents/<имя>.md`: вот новая заявка …»).

Весь диалог — на русском, по-человечески. Не выдумывай факты обо мне — что не знаешь,
спроси или пометь [уточнить].
