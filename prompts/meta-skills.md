# Мета-навыки — голос и быстрый скриншот (по желанию)

Предлагай это ПОСЛЕ проверки окружения и знакомства, одним предложением: «Хочешь настроить
две мелочи, которые сильно ускоряют работу — голосовой ввод и скриншот сразу в буфер?».
Не навязывай — оба по желанию.

⚠️ **Границы.** Ты не можешь установить это за владельца: это программы и разрешения его
компьютера. А если ты в облачной сессии — доступа к его маку у тебя нет вообще. Твоя роль —
провести по шагам, нажимает он сам.

## 1. Голосовой ввод (диктовка вместо клавиатуры)

Ставить задачи голосом быстрее, чем печатать. Под macOS два пути:

- **Проще всего — Wispr Flow (платный).** Скачать с официального сайта, установить, дать
  доступ к микрофону и «Универсальному доступу» (Accessibility), назначить клавишу-зажим.
  Работает из коробки, почти нечего настраивать. Новичку рекомендуй именно его.
- **Бесплатно — Handy (или аналог).** То же самое, но по настройкам заметно больше возни.
  Отдельный локальный Whisper (whisper.cpp / mlx) — только для продвинутых, новичку не надо.

> Ставь только с официального сайта приложения — сверь ссылку, не с случайного зеркала.

**Проверка:** зажал клавишу, сказал фразу — текст напечатался в чате.

## 2. Скриншот сразу в буфер обмена

macOS умеет это сам — **отдельное приложение НЕ нужно.**

- **Готовое сочетание:** `Cmd + Shift + Ctrl + 4` → выдели область → картинка уже в буфере
  (`Ctrl` = «в буфер, а не в файл»). Вставляй `Cmd + V` куда угодно.
- **На свою клавишу:** System Settings → Keyboard → Keyboard Shortcuts → Screenshots →
  строка «Copy picture of selected area to clipboard» → назначь своё сочетание. Предупреди:
  одиночные вроде `Cmd+3` часто заняты в приложениях — лучше сочетание без конфликта.
- **Чтобы и обычные `Cmd+Shift+3/4` шли в буфер, а не в файл:** `Cmd+Shift+5` → Options →
  Save to → Clipboard.

**Проверка:** нажал сочетание, выделил область, `Cmd + V` — картинка вставилась.
