# Базовые скиллы для агентов

Скилл — это готовое умение, которое агент подключает под задачу (в отличие от простого
текста-инструкции). Ниже — те скиллы, что чаще всего пригодятся новичку. Ставить всё сразу
не нужно: подключай по мере надобности.

## Веб-поиск и действия в браузере

- **Свой браузер агента (Playwright).** Агент открывает отдельный, чистый браузер, ходит по
  сайтам, ищет, читает страницы, заполняет формы, делает скриншоты. Это его «руки в вебе».
  Твои личные вкладки и логины он не трогает — браузер отдельный.
- **Действия в твоём браузере (Chrome-расширение) — по желанию.** Если задача именно в том,
  чтобы что-то сделать в твоём уже залогиненном браузере (например, там, где ты авторизована
  и агенту не нужно вводить пароли), ставится отдельное расширение для Chrome. Нужно не всегда —
  для обычного поиска хватает браузера агента.

## Генерация картинок

Два пути, выбирай по тому, что у тебя есть:

- **Есть подписка ChatGPT/Codex** → скилл **codex-image** (он уже лежит в стартере, в
  `.claude/skills/codex-image/`). Картинки генерируются через твою подписку — без отдельного
  API-ключа и без оплаты за каждую картинку. Один раз нужно войти командой `codex login`
  (агент подскажет, если не залогинена).
- **Подписки нет** → ключ **Gemini**. Зайди на `aistudio.google.com` → «Get API key» → скопируй
  ключ и положи его в `.env` строкой `GEMINI_API_KEY=...`. Настоящий ключ — только в `.env`,
  в код и в git он не попадает.

## Важная оговорка: аналитика и данные — это НЕ скилл

Подтянуть цифры из соцсетей, CRM, таблиц или другого сервиса — это не скилл, а **отдельный
коннектор** (через MCP или API этого сервиса). Он подключается отдельно, обычно с ключом
доступа. Не путай: скилл — это умение агента, коннектор — это труба к твоим данным.
