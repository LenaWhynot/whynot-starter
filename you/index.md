# you/ — кто ты

Карта папки. Агент читает этот файл первым, остальные — по надобности.

| Файл | Что внутри | Заполнен? |
|------|-----------|-----------|
| `profile.md` | Кто ты: бэкграунд, сильные стороны, текущая точка | ⬜ заполняется интервью |
| `voice.md` | Как ты пишешь и говоришь: тон, любимые слова, стоп-слова | ⬜ заполняется интервью |
| `feedback.md` | Уроки, которые агент выучил о тебе (растёт сам) | пополняется в работе |

Пустой файл — не страшно. Агент спросит, когда информация понадобится, или предложи
ему интервью: `prompts/interview.md`.
